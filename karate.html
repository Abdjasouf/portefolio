Projet Karaté

Objectif du projet :

Concevoir une application de gestion de compétitions de Karaté (kata) dans le contexte de la M2L, en utilisant Visual Studio (C#), une base de données MySQL, et ADO.NET pour les interactions.

Outils utilisés :

  • Langage : C#
  • Base de données : MySQL
  • Logiciel : Visual Studio Enterprise 2022

Étapes attendues :

  • Ajouter
  • Modifier / Supprimer
  • Consulter / Afficher

Mes missions :

T1.1 Gestion des membres : Cette tâche doit permettre la création, la modification et la suppression d’un membre à partir d’un formulaire. Il faudra réaliser la maquette de ce formulaire, le faire valider par le chef de projet avant de commencer le codage.

T1.4 Consultation des licences : Cette tâche doit permettre l’affichage de la licence.
