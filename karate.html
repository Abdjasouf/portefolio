Projet Karaté

Contexte :

Projet de gestion de compétitions de karaté dans le cadre de la M2L,

Objectif du projet :

Développer une application permettant de gérer les membres et consulter les licences.

Outils et architecture :

  • Langage : C#
  • Framework : .NET
  • Base de données : MySQL
  • Accès données : ADO.NET

Mes missions :

  • Création du formulaire de gestion des membres
  • Développement des fonctionnalités CRUD
  • Affichage des licences
  • Tests et validation des fonctionnalités

Compétences acquises :

  • Manipulation de base de données
  • Conception d’interfaces utilisateur
