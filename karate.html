Projet Restaurant

Contexte :

Projet de développement d’une application web permettant aux utilisateurs de rechercher et d’interagir avec des restaurants.

Objectif du projet :

Ajouter des fonctionnalités dynamiques à un site existant pour améliorer l’expérience utilisateur.

Outils et architecture :

  • Langages : PHP, HTML/CSS
  • Base de données : MySQL
  • Architecture : MVC

Mes missions :

  • Ajout de fonctionnalités de recherche
  • Gestion de la connexion utilisateur
  • Implémentation du système de notation
  • Développement des interactions (like, commentaires)

Compétences acquises :

  • Gestion des utilisateurs
  • Requêtes SQL dynamiques
  • Développement d’interfaces interactives
  • Logique métier côté serveur
