Projet "Le P'tit Jardinier"

Contexte :

Le P’tit Jardinier est une application web conçue pour une entreprise de paysagisme experte dans l’entretien des espaces verts et la taille de haies.

Objectif du projet :

Développer une application pour la société "Le P'tit Jardinier". permettant de gérer les utilisateurs, les devis et les droits d’accès.

Outils et architecture :

  • Framework : Symfony 7
  • Langage : PHP (POO)
  • ORM : Doctrine
  • Base de données : MySQL
  • Sécurité : Authentification + rôles (ROLE_USER / ROLE_ADMIN)

Mes missions :

  • Création des entités et gestion de la base de données
  • Mise en place du système d’authentification
  • Gestion des rôles et des accès utilisateurs
  • Développement des fonctionnalités CRUD

Compétences acquises :

  • Connaissance du framework Symfony
  • Gestion des utilisateurs et sécurité web
  • Utilisation d’un ORM (Doctrine)
  • Structuration d’une application professionnelle
