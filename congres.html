Projet Congrès

Contexte :

Application web de gestion du congrès national des Maisons des Ligues (ANALIM) — gestion des inscriptions, sessions, activités, facturation et règlements. Réalisé en PHP orienté objet (MVC) avec une base MySQL.

Outils et architecture :

  • Langage : PHP (POO)
  • Architecture : MVC (Controllers / Models / Views)
  • Base de données : MySQL (modélisation MCD → MLD)
  • Accès base : PDO
  • Gestion de version : Git

Objectifs de la mission 4 (T1.5, T2.4) :

  • T1.5 — Gestion des organismes payeurs : CRUD complet (Créer, Modifier, Supprimer, Lister) des organismes : entreprises, administrations, universités, lycées.
  • T2.4 — Règlements : Enregistrement du règlement d'une facture existante, association d’un organisme payeur si nécessaire, impression d’un ticket récapitulatif (montant, facture, payeur, congressiste) et possibilité d’annuler le règlement.
