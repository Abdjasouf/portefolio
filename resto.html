Projet Restaurant

Objectif du projet :

Compléter un site web de gestion de restaurants déjà partiellement développé, en y ajoutant plusieurs fonctionnalités pour les utilisateurs connectés.

Outils utilisés :

  • Langages : PHP, HTML/CSS
  • Base de données : MySQL
  • Logiciels : Visual Studio Code, XAMPP

Fonctions ajoutées :

  • Recherche de restaurants (nom, adresse, type de cuisine)
  • Connexion / Déconnexion d’un utilisateur
  • Aimer / ne plus aimer un restaurant
  • Noter et commenter un restaurant

Mes missions :

• Ajout et modification de contrôleurs (recherche, connexion, aimer, critiquer)
• Création de fonctions PHP dans les modèles (requêtes SQL avec filtres dynamiques)
• Intégration de vues conditionnelles selon l’état de connexion de l’utilisateur
• Gestion des interactions utilisateur (aimer, noter, critiquer)
