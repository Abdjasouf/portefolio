Stage 1 - Broussaud Textile

Lieu : Les Cars (Nouvelle-Aquitaine)

Date : 26 Mai 2025 – 26 Juin 2025

Maître de stage : CANE Alexandre

Poste : Développeur

Objectif du projet

Développement d’une application web Symfony permettant de récupérer des données sportives via une API externe, de les stocker en base de données MySQL et de les exploiter via une interface web.

  • Récupération API football
  • Stockage en base MySQL
  • Affichage via Symfony + Twig

Environnement technique

  • Symfony
  • PHP 8.2
  • MySQL
  • Doctrine ORM
  • VS Code

Bilan

Ce stage m’a permis de renforcer mes compétences en Symfony, Doctrine et gestion de bases de données. J’ai également appris à structurer une application complète avec une architecture MVC.
