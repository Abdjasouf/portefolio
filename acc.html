Projet "Aux Claviers Citoyens"

Contexte :

Mission de développement d'une solution numérique pour la pratique du e-sport en environnement militaire.

Objectif du projet :

Développer une application mobile connectée à une API sécurisée, permettant de gérer des tournois e-sport, les équipes participantes et le suivi des performances en temps réel.

Outils et architecture :

  • Plateforme : Application mobile (consommation API REST)
  • Backend : API REST sécurisée
  • Authentification : OAuth2 (Bearer Token)
  • Architecture : Client / Serveur
  • Collaboration : BTS SIO SISR

Mes missions :

  • Développement du module de gestion des équipes
  • Implémentation d’un CRUD complet (création, modification, suppression)
  • Consommation de l’API via des requêtes sécurisées
  • Affichage dynamique des équipes (ID, nom, score)
  • Participation à la conception de l’interface mobile

Compétences acquises :

  • Gestion de l’authentification sécurisée (Token)
  • Travail en équipe multi-projets
  • Développement d’interfaces dynamiques
  • Compréhension d’une architecture distribuée
