Projet "Aux Claviers Citoyens"

Contexte : "Aux Claviers Citoyens"

Mission de développement d'une solution numérique pour la pratique du e-sport en environnement militaire. L'objectif est de fusionner compétition virtuelle et effets réels via des objets connectés, tout en renforçant la cohésion et le leadership des troupes.

Environnement Technique :

  • Plateforme : Application Mobile (Consommation d'API REST)
  • Backend / API : Authentification OAuth2 (Bearer Token)
  • Collaboration : Interopérabilité avec les BTS CIEL (IoT) et SIO-SISR (Infrastructure)
  • Localisation : Phase finale prévue sur la Base Aérienne 721 de Rochefort

Ma Mission : Gestion des Équipes

J'ai été responsable du module de gestion des compétiteurs. Ce module est crucial pour structurer les tournois et assurer le suivi des performances.

Fonctionnalités développées :

  • CRUD Équipes : Création, modification (nomination) et retrait d'équipes dans un tournoi spécifique via l'API.
  • Consommation de l'API : Implémentation des requêtes sécurisées pour lister les équipes et leurs points accumulés.
  • Affichage Dynamique : Visualisation précise des détails d'une équipe (ID, Nom, Score) pour les organisateurs.
  • Expérience Utilisateur (Front-end) : Design de l'interface mobile pour une gestion fluide sur le terrain.

Défis relevés :

  • Sécurité : Intégration du jeton d'accès (Passport numérique) pour chaque requête API.
  • Travail d'équipe : Coordination avec les développeurs des modules "Matchs" et "Tournois" pour garantir la cohérence des données.
